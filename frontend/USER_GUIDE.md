# 前端使用手册

## 1. 先准备什么

使用前需要准备：

- 前端页面地址
- 后台接口可正常访问
- `X-API-Token`

如果没有 `X-API-Token`，页面无法查询数据。

## 2. 首页

首页用于选择新闻列表区域并生成规则。

操作步骤：

1. 输入新闻列表页 URL
2. 输入 `X-API-Token`
3. 点击“访问”
4. 在左侧预览区点击新闻列表区域
5. 点击“生成规则”
6. 在右侧查看预览结果
7. 如果结果正确，点击“确认成功”

## 3. 站点管理

站点管理页用于查看和维护站点。

可以做的事情：

- 查询站点
- 查看站点详情
- 编辑站点名称、状态、备注
- 删除站点

说明：

- 删除站点后，相关版本、运行记录和文章也会一起删除

## 4. 新闻列表

新闻列表页用于查看已经抓取到的文章。

可以按这些条件筛选：

- 关键词
- 标题
- 站点 ID
- 运行 ID
- 发布时间范围

说明：

- 点击标题可以直接打开原文

## 5. 常见问题

### 看不到数据

先检查：

- `X-API-Token` 是否正确
- 后台服务是否已经启动
- 接口地址是否配置正确

### 预览失败

通常是以下原因：

- 目标页面无法访问
- 选择的列表区域不对
- 后台运行环境异常
